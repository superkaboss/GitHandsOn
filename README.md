# GitHandsOn

1. リモートリポジトリを作成する
    - readme.mdを有効にしておく
    - 作業場所はGitHub
2. リモートリポジトリをcloneする
    - 作成したリモートリポジトリのGitHubのCodeからHttpsのURLをコピー
    > git clone URL
3. 作業するブランチを作成する
    > git switch ブランチ名
4. ファイルの変更
    - 記載を増やす
5. 変更のステージ
    - git add .
6. commit
    - git commit -m "ハンズオンのcommitまでの手順を記載
